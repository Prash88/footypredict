# React Webpack Bootstrap Heroku Boilerplate

### Stack

* React
* Babel
* Webpack
* iojs
* heroku-buildpack-webpack

### Build
```
npm run build
```

### Dev
```
npm run dev
```

### Heroku
```
heroku create
heroku config:set BUILDPACK_URL=https://github.com/ddollar/heroku-buildpack-multi
git push heroku master
```
